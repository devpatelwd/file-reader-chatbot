# file-reader-chatbot
AI chatbot that answers questions about uploaded text file

## Built with
- python
- Google gemini api
- streamlit

## How to run
1. Clone the repo
2. Install dependencies : first - pip install -q -U google-genai 
                          Then  - pip install streamlit
3. Add your Gemini Api Key to .env file
4. Run : streamlit run app.py
